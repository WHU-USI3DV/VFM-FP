# Owner Decisions

This document records the project-owner decisions used for the current VFM-FP public release candidate.

## Release Package Decision

Decision: use the lean release profile for the final public package.

Current candidates:

| Folder | Purpose | Current status |
| --- | --- | --- |
| `release/VFM-FP-open-source` | Full source-only release generated by `tools/prepare_release.ps1`. | Includes Keep, Review, and Optional files. |
| `release/VFM-FP-essential` | Local source-only review copy used during cleanup. | Same scope as open-source in the current workspace. |
| `release/VFM-FP-lean-proposed` | Compact release generated by `tools/prepare_lean_release.ps1`. | Includes Keep and Review files, excludes Optional files. |
| `release/VFM-FP-final` | Final source-only package generated from the lean profile. | Intended final upload candidate. |

## Directory Naming Decision

Decision: rename the segmentation code directory to `VCFS`, short for VFM-CNN fusion segmentor.

## License Decision

Decision: do not add a root project `LICENSE` in this cleanup pass.

The upstream DeepLabv3+ baseline notice remains preserved at `VCFS/LICENSE`.

Important note: publishing without a root license means the project does not grant broad reuse rights beyond notices already present in third-party code. Revisit this before public GitHub release if explicit reuse terms are needed.

## Citation Decision

PDF metadata confirms:

- Title: Leveraging Visual Foundation Model Priors for Facade Parsing from Street View Images
- Authors: Zhe Chen; Qingwen Tan; Fuxun Liang; Chen Long; Pangyin Li; Zhiming Liu; Junjie Chen; Zhen Dong

Still pending from official publisher metadata:

- Year
- Journal volume, issue, and pages
- DOI

Decision: keep `docs/citation_template.md` as the draft and do not add a root `CITATION.cff` until official metadata is available.

## Review Files Decision

Decision: keep all six Review files in the final lean package.

| File | Reason to keep |
| --- | --- |
| `VCFS/.gitignore` | Harmless submodule hygiene if the segmentation module is reused independently. |
| `VCFS/benchmark.py` | Allows optional reproduction of model-complexity/FPS statistics. |
| `VCFS/README.md` | Preserves upstream DeepLabv3+ baseline context. |
| `VCFS/requirements.txt` | Preserves original baseline dependency reference. |
| `VCFS/utils/dataloader_latest_ecp.py` | Keeps ECP-specific training support available. |
| `SDA/diffusion/voc_annotation.py` | Keeps SDA split-generation helper available. |

## Data And Weights Decision

Decision: publish source code only.

The release candidates intentionally exclude datasets, generated images, checkpoints, model caches, and archives. Add external download links later only after dataset and trained-weight redistribution rights are confirmed.
